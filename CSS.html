Most developers and programmers know that CSS files are integral to a more attractive website. I love to deal with CSS because I find my creative side to flourish when designing web pages. I love the process of developing and crafting animations, fonts and making sites as aesthetically pleasing as possible.

For instance, I implemented various functionalities, animations, and styles on this website alone. Developing this portfolio page was an entertaining experience to watch barren HTML files flourish into artful, almost professional-looking web pages. Some of the previously mentioned functionalities include the font, the animation on the 'Skills' page, and the headers and footers. To elaborate, I accessed the font using a Google API (application programming interface) because I went on their 'Google Developers' page and wanted to use fonts specific to Google. Moreover, the animation from my 'Skills' page came about because I wanted to add a dynamic element to the logos. And finally, the footers and headers I made were something that I wanted to add to introduce a more sleek look to the entire site.
