JavaScript. It is such as robust language in terms of capabilities. I have it on the lower end of the list of programming skills because it is certainly not up to par with other languages I know such as Python, MATLAB, HTML, or CSS (in that particular order, actually). Nevertheless, in between this site and studying JavaScript, I familiarized myself with VanillaJS. To those wondering, especially those outside the software developer community, VanillaJS simply refers to coding in JavaScript without any additional libraries or frameworks such as Ajax or jQuery.

All in all, my projects include smaller-scale endeavors such as calculators, games, login forms, and clones of pre-existing sites/apps such as Netflix, Playstation 4 UI (user interface), and Instagram. Moreover, I am familiar with several of JavaScript's libraries such as jQuery and React. In fact, this website utilizes jQuery. And all of the previously mentioned clone-projects were developed using React.
